# natsumiWaveAudioIndicator
Wave Audio indicator for natsumi browser

js script either assigns a wave color per defined site or extracts the dominant color from the favicon if its not defined.

Usage instructions:

userChrome.css on your chrome folder root for your profile (about:profiles => Open profile folder => chrome)
dynamicWaveColor.uc.js on the natsumi scripts folder (about:profiles => Open profile folder => chrome => natsumi => scripts)
wave.svg on chrome resources folder (about:profiles => Open profile folder => chrome => resources => assets)
